# Default Theme for civisions.site #

Default theme for http://cvisions.site

### Installation

## Node and npm

Install node and npm from https://nodejs.org/en/

## Install Gulp

Install gulp cli


```
npm install -g gulp-cli
```



### Build Instructions

Run the following command to build after making any changes to src files



```
gulp build
```



Run the following to build and host with auto live refresh and recompile feature. It will actively check for changes and automatically recompile files and refresh page


```
gulp
```



Visit localhost:8080 after hosting via `gulp` command
